1.1多que es una URL?

una URL es una cadena de texto que especifica donde se puede encontrar un recurso en internet

1.2多que es una URL absoluta?

una url absoluta te proporciona una URL completa del documento junto al protocolo (http://)

1.3多que es una URL relativa?

las URL relativa describen la ruta a un archivo relativo al actual documento

1.4 多cual es la diferencia entre una URL relativa y una URL absoluta?

una URL absoluta te puede proporcionar la url completa mientras que la relativa solo describe la ruta a un archivo al actual documento
